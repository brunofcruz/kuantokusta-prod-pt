API de listagem de Produtos

## Description

Esta API foi criada para o teste técnico da Kuanto Kusta PT e tem como finalidade exibir uma lista fictícia de produtos.
Para sua criação foram utilizados:
NestJS e Mongoose

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
